# AI Business Coach App

A powerful AI-powered business coaching application designed for entrepreneurs and solopreneurs. Get personalized guidance on business strategy, leadership development, metrics, scaling, and overcoming specific challenges.

## Features

### Core Functionality
- **Intelligent Chat Interface**: Chat with an AI coach that understands business context
- **Voice Input/Output**: Use voice commands for hands-free coaching sessions (Web Speech API)
- **Conversation History**: All coaching sessions are saved and can be revisited
- **Context-Aware Responses**: The AI remembers your industry, business stage, and challenges
- **Quick Topic Starters**: Jump into key business topics with one click

## Usage

### Getting Started

1. Open `index.html` in a modern web browser
2. Choose a quick-start topic or type your own question
3. The AI coach will provide tailored guidance with frameworks and actionable advice

### Example Questions

- "How do I measure product-market fit?"
- "Help me with time management as a founder"
- "What metrics should I track for my SaaS business?"
- "I'm struggling with imposter syndrome"
- "How do I know when to scale my business?"
- "What's the best way to set goals for my team?"

### Voice Input

Click the microphone button to use voice input. The app will transcribe your speech and provide a response. Voice input requires a browser that supports the Web Speech API (Chrome, Edge, Safari).

## Technical Details

### Tech Stack
- **HTML5**: Semantic markup and structure
- **CSS3**: Modern responsive design with CSS Grid and Flexbox
- **Vanilla JavaScript**: No frameworks required
- **Web Speech API**: Voice recognition and synthesis
- **LocalStorage**: Client-side data persistence

- `index.html` - Main application structure

- `knowledge-base.js` - Comprehensive business coaching knowledge
- `app.js` - Application logic, AI responses, voice integration
